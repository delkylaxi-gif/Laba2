# Перемножение матриц 4096×4096: сравнение трёх алгоритмов

## Описание

Программа выполняет перемножение двух квадратных матриц размера **4096×4096** с элементами типа `float`. Реализованы три варианта алгоритма с оценкой производительности в **MFlops** и сравнением эффективности.

---

## Алгоритмы

### Вариант 1 — классический (по формуле из линейной алгебры)
Реализует прямое вычисление по определению произведения матриц:
C[i][j] = Σ(k=1..n) A[i][k] · B[k][j]
Три вложенных цикла `(i, j, k)` без оптимизаций.

### Вариант 2 — библиотечный (cblas_sgemm из Intel MKL)
Использует высокооптимизированную реализацию BLAS (Basic Linear Algebra Subprograms) из пакета Intel MKL. Учитывает:
- Кэширование данных
- Векторные инструкции (AVX2/AVX-512)
- Многопоточность (OpenMP)
- Блочные алгоритмы

### Вариант 3 — оптимизированный студентом
Реализует следующие оптимизации:
1. **Смена порядка циклов** — переход от `(i, j, k)` к `(i, k, j)` для улучшения локальности данных (использование строк вместо столбцов)
2. **Пропуск нулевых элементов** — исключение умножений при `A[i][k] == 0`
3. **Минимизация индексации** — вынос повторяющихся индексов во временные переменные

---

## Шаги работы программы

1. Генерация двух случайных матриц **A** и **B** размера 4096×4096 (значения от -10.0 до 10.0)
2. Вычисление произведения матриц тремя способами с замером времени
3. Оценка сложности по формуле:  
   `c = 2 · n³` операций (каждое умножение и сложение)
4. Расчёт производительности в MFlops:  
   `p = c / t · 10⁻⁶`
5. Проверка корректности (сравнение вариантов 1 и 3 с эталонным вариантом 2)
6. Сравнение производительности и проверка условия:  
   **вариант 3 ≥ 30% от варианта 2**
